# GitHub User Finder

**Автор:** [Закиров Фаниль Радикович]

## Описание
Графическое десктоп-приложение для поиска пользователей GitHub через официальный REST API. Приложение позволяет вводить поисковый запрос, отображать найденные ники в списке, добавлять выбранных пользователей в локальное избранное и сохранять список в файл `favorites.json`. Интерфейс не блокируется во время сетевых запросов.
